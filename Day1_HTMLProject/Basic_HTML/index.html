Frontend Development Roadmap

Learn the technologies used for building modern websites.

What is Frontend Development?

Frontend Development is the development of the graphical user interface (GUI) of a website, through the use of HTML, CSS, and JavaScript, so that users can view and interact with that website.

Core Technologies

  • HTML: The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

    HTML is used to structure content and create elements like paragraphs, links, images, and forms.

  • CSS: Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML.

    1. Pure CSS

    2. Bootstrap (Framework)

    3. Tailwind (Framework)

    4. Sass (Preprocessor)

  • JavaScript: JavaScript, often abbreviated as JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It is used to add functionality to websites.

    1. Vanilla JS (Pure JavaScript)

    2. jQuery (Library)

    3. React (Framework)

    4. Angular (Framework)

    5. Vue (Framework)

    6. Typescript (Preprocessor)

Example Code: Basic HTML Structure

Here is an example of a basic HTML structure:

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>My Webpage</title>
    </head>
    <body>
        <h1>Welcome to My Website</h1>
        <p>This is an example of a basic HTML page.</p>
    </body>
</html>

Frontend Frameworks & Libraries

Frontend frameworks and libraries help speed up development by providing pre-written code and structure.

  • React: A JavaScript library for building user interfaces.
  • Vue: A progressive JavaScript framework for building modern web applications.
  • Angular: A TypeScript-based framework for building single-page applications.
  • Bootstrap: A CSS framework for building responsive websites.
